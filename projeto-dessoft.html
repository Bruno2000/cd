Trabalho final - Design de Software

Descrição

O trabalho final de Design de Software consiste em um desenvolvimento de um jogo por meio de programação em Python, usando a biblioteca Pygame.

Competências

  • Capacidade lógica
  • Programação em Python
  • Trabalho em equipe
  • Criatividade
  • Capacidade de abstração

Passo a passo

  1. Mockup da ideia;
  2. Desenho das telas do jogo;
  3. Implementações em python;
  4. Criação de um menu;
  5. Uploadar no GitHub;
  6. Tornar o modelo demonstrável para a feira.
